# 关于使用Xcode调试
查找和排查代码问题是开发过程中的很重要的步骤。Xcode调试器内置强大功能来进行通用调试，并在你的应用程序加载的时候自动运行。调试器能够帮助你：  

* 确定和定位问题
* 检测运行代码的控制流和数据结构来找到原因
* 根据你的代码设计解决方案和编辑
* 运行修改后的应用程序并确保它的修复能够运行
