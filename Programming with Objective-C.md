# 关于OC
在为OS X和iOS编写软件时，OC是一门主要使用的编程语言。这是一个C语言的超集，提供面向对象的功能以及动态运行时能力。OC继承了C语言的语法、原始数据类型以及流控制语句，并且添加了典型的类和方法语法。它还为对象的管理提供了语言级别的支持，并且在动态类型绑定时提供了对象字面量，将很多职责推迟到了运行时。
