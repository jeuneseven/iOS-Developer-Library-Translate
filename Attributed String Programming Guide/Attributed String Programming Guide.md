## 属性字符串编程指南简介

属性字符串编程指南描述了属性字符串对象，实例化的 NSAttributedString 类或者 CFAttributedString 的 Core Foundation 不公开类型，它管理着文字属性集合，比如字符间距，字体以及字符串相关或者单个字符。

如果你直接操作属性字符串对象的话，你就应该阅读本文档。
